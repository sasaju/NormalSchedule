# 河大课表(Android)

# 本项目停止运行和维护

## 简介

针对河北大学的教务系统编写的课程表APP，支持无校园网环境下一键导入个人课程。欢迎大佬PR

## 数据来源及其责任相关

数据来源：*河北大学本科生教务系统。*\
如有违规侵权行为，请邮件(lifly@lifly.cn)联系本人，本人将第一时间停止违规侵权行为。
